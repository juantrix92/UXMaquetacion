Cómo dejar comentarios

En cada prototipo hay un botón flotante “+”. Al pulsarlo, se abre el panel para escribir comentarios por pantalla. Se guardan en tu navegador (demo con localStorage).

Incluye

  • Flujos: Home, Login, Tablero, Tareas, Detalle, Crear, Notificaciones, Ajustes.
  • Grid 12 col (web) / 4 col (mobile), capas de comprensión y micro‑interacciones.
  • Estructura lista para GitHub Pages.

Publicar en GitHub Pages

  1. Sube esta carpeta /site a un repositorio y renómbrala como /docs en el repo.
  2. En Settings → Pages, usa Deploy from a branch y elige main /docs.
  3. Tu URL será https://<usuario>.github.io/<repo>/.
